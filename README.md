# Getting started

- `npm init stencil` > component > name: `stencil-example` > confirm
- `cd stencil-example`
- `npm i`
- `npm start`
- Open `dev server: http://localhost:****/` should see;
> `Hello, World! I'm Stencil 'Don't call me a framework' JS`
